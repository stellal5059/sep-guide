Github



What is Github?

Github is a social media site for coders, instead of sharing pictures and statuses coders can share projects and ideas



Git

Git is a version control system used for software development. Mainly being used in the command line (or the terminal) that is used to navigate through files and folders with little commands.

Using Github

Using github isn't difficult, to start using github you would first make an accoutn on github.com. Then you link your IDE with your account by following the directions given to you.

Why Github is useful

Github has many benefits such as collaboration and communication. With github you can collaborate and communicate with coders all across the world, these people can help with any bugs in your code or any problem that comes up. Also you can collaborate with friends and work on projects together.
