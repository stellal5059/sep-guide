What H.C.B stands for?

  • CSS stands for Cascading Style Sheet and it's used in order to make websites and webpages look pretty with many colors and even sometimes animations.

  • Bootstrap is another component of HTML and CSS that uses it's own CSS library of classes to help organize the programmer's webpage and make the context look neat.
